Welcome to Ascendia:¶

Bienvenue dans l’univers d’Ascendia, un serveur Minecraft où aventure, stratégie et communauté se rencontrent !

Cette documentation a pour but de vous guider dans votre expérience sur nos serveurs Faction et Skyblock, en vous expliquant les règles, systèmes et fonctionnalités que nous proposons.

Nous vous recommandons vivement de lire attentivement chaque section pour garantir une expérience de jeu fluide et respectueuse pour tous.

📌 Besoin d’aide ou envie de discuter avec la communauté ? Rejoignez notre serveur Discord : https://discord.gg/RyJvUQNZre Nos équipes y sont présentes pour répondre à vos questions, organiser des événements et vous tenir informé des dernières nouveautés.

Bonne lecture et bon jeu sur Ascendia !

1. Règles Générales Ascendia:¶

Ces règles s’appliquent à tous les membres de la communauté Ascendia, que ce soit sur le serveur Faction ou Skyblock.

1. Comportement Généra:¶

  • Comportement Respectueux : La discrimination, les propos haineux ou le contenu offensant sont strictement interdits. - Exemple : utiliser des insultes ou des propos dégradants dans le chat.

  • Pseudos et Skins Appropriés : Les noms d’utilisateur, skins et capes doivent rester appropriés. - Exemple : un skin contenant des images explicites ou offensantes.

  • Pas de Publicité : La publicité pour d’autres serveurs ou services est interdite. - Exemple : partager un lien vers un autre serveur Minecraft dans le chat.

  • Pas de Spam ou Flood : Pas d’envoi excessif de messages, de majuscules ou de caractères répétés. - Exemple : écrire "REJOIGNEZ MAINTENANT !!!" plusieurs fois.

  • Pas d’Impersonation : Ne vous faites pas passer pour un membre du staff ou un autre joueur. - Exemple : modifier votre pseudo pour imiter un membre du staff.

  • Pas de Harcèlement : Harceler ou intimider d’autres joueurs est interdit. - Exemple : cibler un joueur avec des propos négatifs en continu.

  • Pas d’Exploitation de Bugs : Exploiter des bugs ou glitches à son avantage est interdit. - Exemple : utiliser un bug de duplication d’objets.

  • Pas de Triche : L’utilisation de mods non autorisés, de hacks ou de macros est interdite. - Exemple : utiliser un auto-clicker pour automatiser les actions.

  • Responsabilité du Compte : Vous êtes responsable de tout ce qui est fait avec votre compte. - Exemple : un ami utilise votre compte pour tricher, vous êtes quand même responsable.

  • Respect du Staff : Suivez les instructions données par le staff. - Exemple : si un membre du staff vous demande d’arrêter un comportement, obéissez.

2. Règles du Serveur Faction:¶

Ces règles s’ajoutent aux règles générales et concernent uniquement le serveur Faction.

1. Territoire et Bases:¶

  • Noms de Factions Appropriés : Pas de noms offensants ou inappropriés. - Exemple : donner à sa faction un nom vulgaire ou discriminatoire.

  • Pas de Claim Abusif : Ne pas entourer une faction adverse pour l’empêcher de s’étendre. - Exemple : revendiquer tous les terrains autour d’une autre faction.

  • Bases Raidables : Les bases doivent pouvoir être raidable. - Exemple : construire une base totalement inaccessible.

  • Pas d’AFK sur les Murs : Ne pas rester AFK sur les murs pour empêcher les raids. - Exemple : rester connecté sans bouger sur un mur de base.

  • Pas de Redstone dans les Claims : Les systèmes redstone dans les claims sont interdits. - Exemple : créer des pièges redstone dans votre base.

2. Combat et Raid:¶

  • Pas de Combat Logging : Se déconnecter en plein combat est interdit. - Exemple : quitter le serveur pour éviter la mort.

  • Pas de Glitch dans les Bases : Entrer dans une base via un glitch est interdit. - Exemple : utiliser une ender pearl pour traverser un mur.

  • Mods Interdits : Les mods donnant un avantage injuste sont interdits. - Exemple : utiliser un x-ray pour repérer des bases.

  • Pas de Factions Alternatives : Créer plusieurs factions pour contourner les règles est interdit. - Exemple : créer une 2e faction pour réclamer plus de territoire.

  • Pas d’Abus de Shield : Abuser du système de protection est interdit. - Exemple : prolonger le temps de shield via un bug.

3. Règles du Serveur Skyblock :¶

Ces règles s’ajoutent aux règles générales et concernent uniquement le serveur Skyblock.

1. Gestion de l’Île:¶

  • Noms d’Îles Appropriés : Les noms d’îles doivent rester corrects. - Exemple : nommer son île avec un mot inapproprié.

  • Warps Sécurisés : Les warps doivent être sûrs pour les visiteurs. - Exemple : créer un warp qui piège ou tue le joueur.

  • Pas de Farm AFK : Automatiser une ferme en étant AFK est interdit. - Exemple : utiliser un auto-clicker pour récolter.

  • Pas de Mining AFK : Miner automatiquement sans jouer est interdit. - Exemple : miner des blocs à l’aide d’un script en étant absent.

  • Pas d’Arnaques : Les arnaques lors d’échanges sont interdites. - Exemple : promettre un objet contre de l’argent et ne rien donner.

2. Économie et Échanges:¶

  • Pas de Vente IRL : Échanger des objets en jeu contre de l’argent réel est interdit. - Exemple : vendre des coins contre des euros via PayPal.

  • Pas de Boost de Profil : Utiliser un autre profil pour booster son compte principal est interdit. - Exemple : transférer les objets d’un alt vers votre île principale.

  • Pas de Jeux de Hasard : Les activités de type casino sont interdites. - Exemple : ouvrir un casino sur votre île.

  • Pas de Macros/Scripts : L’automatisation par scripts est interdite. - Exemple : utiliser un script pour pêcher en boucle.

  • Pas d’Exploitation de Bugs : Toute exploitation de bugs est interdite. - Exemple : duplication d’objets avec un glitch.

4. Règles Hors-Serveur :¶

Ces règles s’appliquent sur les plateformes externes comme Discord.

1. Communication:¶

  • Pas de Spam : Pas de messages répétés ni d’emojis en excès. - Exemple : envoyer 10 fois le même message ou 50 emojis.

  • Pas de Publicité : Ne faites pas la promo d’autres serveurs/services. - Exemple : envoyer un lien Discord vers un autre serveur.

  • Respect de la Vie Privée : Ne partagez pas d’informations personnelles sans consentement. - Exemple : poster le vrai nom ou l’adresse de quelqu’un.

  • Pas de Harcèlement : Aucun harcèlement dans les messages ou publications. - Exemple : envoyer des messages menaçants à un membre.

  • Respectez les Règles des Plateformes : Suivez les CGU de Discord ou autre plateforme utilisée. - Exemple : enfreindre les règles officielles de Discord.

5. Remarques Finales:¶

  • Évolution des Règles : Les règles peuvent être modifiées. Toute mise à jour sera annoncée sur Discord.

  • Faites Preuve de Bon Sens : Si quelque chose semble abusif, ne le faites pas. En cas de doute, demandez au staff.

  • Amusez-vous ! : Nous souhaitons offrir un espace fun et sécurisé pour tous.

Merci de faire partie de la communauté Ascendia !

2. Conditions Générales d’Utilisation – Serveur Discord Ascendia:¶

Dernière mise à jour : 29 mai 2025 Entrée en vigueur dès la connexion au serveur

1. Acceptation des Conditions:¶

En rejoignant le serveur Discord Ascendia, vous acceptez :

  • Ces Conditions Générales d’Utilisation

  • Le règlement interne d’Ascendia (voir section 6)

  • Les Conditions d’utilisation de Discord : https://discord.com/terms

  • Les Règles de la communauté Discord : https://discord.com/guidelines

Toute infraction peut entraîner un avertissement, une sanction temporaire ou un bannissement définitif.

2. Conditions Utilisateurs – AscendiaBot & AaiCS:¶

AscendiaBot est le bot officiel du serveur Ascendia, développé par la Dev Team Ascendia, propulsé par Ascendia AI Core Services (AaiCS).

En interagissant avec AscendiaBot:

  • Vous acceptez que vos messages/commandes puissent être analysés, modérés et enregistrés temporairement.

  • Les réponses de l’IA ne remplacent pas un avis professionnel (juridique, médical, etc.).

  • Certaines fonctionnalités peuvent être instables ou expérimentales.

  • Vous vous engagez à ne pas contourner les limitations, exploiter les failles ou spammer les services.

3. Code de Conduite de l’IA:¶

L’utilisation d’Ascendia AI 4.5, Ascendia e3 mini, et de tous les modules AaiCS est soumise à un usage responsable.

Vous vous engagez à :

  • Ne pas utiliser l’IA pour générer ou promouvoir : - Propos haineux, contenu NSFW, racisme, diffamation ou activités illégales - Discrimination ciblée ou harcèlement

  • Ne pas manipuler ou provoquer l’IA à produire des contenus sensibles ou dangereux.

  • Respecter les limites d’utilisation : fréquence, durée de session, accès Premium.

  • Ne pas utiliser l’IA à des fins commerciales ou frauduleuses.

Les abus détectés par Ascendia Moderation AI ou les modérateurs seront sanctionnés immédiatement.

4. Données et Confidentialité:¶

  • Nous ne vendons ni ne partageons vos données personnelles.

  • Vos interactions peuvent être temporairement enregistrées à des fins de modération ou d’analyse.

  • Vous pouvez demander l’accès ou la suppression de vos données via un ticket officiel.

5. Services Premium:¶

Certaines fonctionnalités sont réservées aux utilisateurs Premium :

  • Accès prioritaire à Ascendia AI 4.5

  • Accès illimité à Ascendia e3 mini

L’abonnement Premium :

  • Est non remboursable sauf indication contraire

  • Peut évoluer avec le temps

  • Ne doit pas être partagé ou revendu
